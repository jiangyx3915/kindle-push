# kindle-push

> 一款基于Electron + Vue开发的Kindle个人文档客户端程序

## 构建方式

### Build Setup

``` bash
# 安装依赖
npm install / yarn

# 启动热加载服务在localhost:9080
npm run dev

# 应用打包
npm run build
```
